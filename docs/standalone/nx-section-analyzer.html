사용 방법
  1. 이미지를 캔버스에 드래그&드롭
  2. 기준선 설정 클릭 → 실제 길이 아는 두 점 클릭 → mm 입력
  3. 측정선 그리기 클릭 → 적층부 두 점 클릭 (수직 스냅, Shift=자유)
  4. 자동 검출된 레이어를 확인 → Excel 내보내기
